\doxysubsection{Descripción detallada}
Representa un cluster de procesadores. 

Representa un cluster con un número de procesadores variable, distribuidos en forma de árbol binario, con identificadores únicos. El primer procesador puede ser cualquiera de ellos. Cada procesador puede tener dos, uno o cero procesadores sucesores, también llamados auxiliares.

El cluster puede añadir, eliminar y reorganizar los procesos de los procesadores ademas de simular el avance del tiempo
